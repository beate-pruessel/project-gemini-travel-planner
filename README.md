# Gemini Travel Planner

Ein minimalistischer, serverloser und komplett kostenloser KI-gestützter Reiseplaner. Dieses Projekt wird vollständig auf GitHub Pages gehostet und nutzt die Gemini 2.0 Flash API nach dem BYOK-Prinzip (Bring Your Own Key).

**Wichtiger Hinweis:** Dieses Projekt befindet sich aktuell in der aktiven Entwicklung und dient vorerst ausschließlich als persönliches Referenzprojekt. Beiträge (Contributions), Pull Requests oder eine aktive Mitarbeit sind zum jetzigen Zeitpunkt nicht erwünscht und werden nicht bearbeitet.

Live-Demo auf GitHub Pages: [https://dein-benutzername.github.io/gemini-travel-planner](https://www.google.com/search?q=https://dein-benutzername.github.io/gemini-travel-planner) (Ersetze dies mit deinem echten Link)

---

## Aktueller Projektstatus (Referenz)

Dieses Repository dient im Moment nur zur Demonstration und als Code-Referenz.

* Das Projekt ist funktionsfähig, wird jedoch von mir als Einzelentwickler fertiggestellt.
* Externe Pull Requests oder Issues zur Code-Optimierung werden bis zur finalen Fertigstellung ignoriert oder geschlossen.

---

## Features

* Komplett kostenlos und serverlos: Kein Backend, keine Datenbanken, keine versteckten Kosten.
* BYOK-Sicherheit (Bring Your Own Key): Dein API-Schlüssel wird ausschließlich in deinem Browser verarbeitet und niemals an Dritte (außer Google) übertragen.
* Maßgeschneiderte Reisepläne: Erstellt personalisierte Routen basierend auf deinem Reiseziel, der Dauer und deinem bevorzugten Reisestil.
* Schlüssel lokal speichern: Option, den API-Schlüssel für zukünftige Besuche sicher im LocalStorage deines Browsers zu sichern.

---

## So erhältst du deinen kostenlosen Gemini API-Key

Falls du die Live-Demo ausprobieren möchtest, benötigst du einen kostenlosen API-Schlüssel von Google:

1. Gehe auf Google AI Studio: [https://aistudio.google.com/](https://aistudio.google.com/)
2. Melde dich mit einem beliebigen Google-Konto an.
3. Klicke auf "Get API Key" und erstelle einen neuen Schlüssel.
4. Kopiere den Schlüssel, füge ihn in das Eingabefeld auf der Webseite ein und starte deine Planung.
